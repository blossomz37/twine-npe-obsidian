Pandoc Conversion Notes

Basic Conversion

pandoc sample.md -o sample.docx

From Notion Export

pandoc notion-export.md -o notion-document.docx

With Custom Styling

pandoc sample.md --reference-doc=template.docx -o styled-output.docx
